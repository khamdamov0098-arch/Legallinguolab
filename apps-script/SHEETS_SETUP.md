## Лист `Teoria test otveti`

Создайте лист с таким именем (или оставьте пустым — Apps Script создаст заголовки сам).

При сохранении записываются: ID, Дата, Имя, Группа, Телефон, Правильно, Всего, Процент, Оценка, Ответы (1-A,2-B…).

## Деплой

1. Вставьте `Code.gs` в Apps Script проекта таблицы.
2. Deploy → Web app → Anyone.
3. URL вставьте в `nav.js` → `SCRIPT_URL`.
